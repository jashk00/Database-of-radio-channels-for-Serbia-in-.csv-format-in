# Database-of-radio-channels-for-Serbia-in-.csv-format-in

Database of radio channels for Serbia in .csv format

This file is a database of radio channels for Serbia in .csv format. It contains VHF and UHF repeaters at 145 MHz and 434/438 MHz, simplex channels (S/SU series) for direct connection, and PMR channels at 446 MHz. For each channel, the frequency, offset, CTCSS/DCS tone, power, and location are listed. Use CHIRP for programming.

За сваки канал наведена је фреквенција, помак (offset), CTCSS/DCS тон,снага и локација.За програмирање користити CHIRP.
